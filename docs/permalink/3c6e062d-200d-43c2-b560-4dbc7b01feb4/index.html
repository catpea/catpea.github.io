On Easily Understanding The Complexity Of Artificial Intelligence Generated Applications

6/14/2026

What I want you to understand, today, is that you need to build, "temporary" satellite applications around your complex AI code base.

You can delete them, but chances are they will become precious, they represent a view of your system through a semi-coherent lens.

I have a vast process library in one of my programming languages, and for grasping that I told AI to create a program like iTunes.

My programming language, has several strong features, compiler directives, a build system with dependency tracking.

And to visualize that, I told AI to create a Control Panel, based on a dashboard.

AI beatifully captured all the features of my language, because it can read the code really well.

All the features, in a single sidebar.

At a glance the New Project Wizard, exposed what I should work on next.

Allow me to clarify, these are not necessary for my language, these beautiful little program can be deleted.

They exist to map features, onto a Graphic User Interface, so that I can see the system better.

I originally told the AI, that just like Text Summarization is Intelligence, satellite applications summarize complex systems.

The reveal extra features that AI added, and they show what needs more work.

And perhaps more importantly, what should be worked on next.

To add satellite applications, you have to think in archetypes, you system has shapes you need to capture and translate them into GUI.

Finally, what I have described to you is an ancient art of taming complexity, it is called Ecological Interface Design.

You may remember it from old movies, big maps on walls with flashing lights.

Those were the old version of abstractions, over complexities like Factories of Phone and Train companies.

The word Ecological does not refer to nature, but the ecology of your system.

The satellite applications, are the new blinking lights.

My approach was created independently of EID, I simply had a need for external views of AI built systems.

I needed apps, but EID revealed that what I was rediscovering, was an old art developed by the original masters of complexity.
