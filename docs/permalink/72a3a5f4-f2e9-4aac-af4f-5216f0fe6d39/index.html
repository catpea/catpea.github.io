The Age Of Code

5/10/2026

I am happy to hear CEOs reporting that AI does not increase productivity, because that means their coders are building their own companies with it.

Long ago, it seems, we needed to make our programs do everything, but today, we have thinking machines, that think, and frighteningly so.

I ran two hard tests, the first one took the top AI 40 minutes to solve, and the tiny errors it made aren’t even worth mentioning.

The resulting programing language Galath Strong Core, I called it Galath AI added Strong core, OK.

Originally the language was named XML, ECMAScript, Signals, but the acronym was too immature.

I did not stop there, I asked AI to create a bootstrap theme with components, crafted for reviewing user interfaces.

And then I created inspectoid where I said, program the bootstrap components into a UI review system with Galath.

I don’t know what is more terrifying, that it worked, or that I knew it would work, and it did not even take very long, minutes.

So I created a powerful language, an application that uses that language, and relies on a UI-kit, a user interface kit that the AI made earlier.

Galath code, is tiny, compared to other programming languages, it is almost a shame how much easier it makes programming web apps.

But making a theme and having AI get everything right in minutes, is so obscenely powerful, that CEOs reporting no gains, should fill us with joy.

Way to go you sneaky cats, I hope your autonomous companies spread to no end.

This still took an entire day, I suspect AI will forever need guidance, it needs a human to go along with it, and ensure right directions.

But if you keep notes, or a skill, the second time around, the AI will problem solve its way through it on its own.

There has been a change in the programming world, but for the better, we can build much larger programs now.

And this isn’t just about making a sale, though one can be made at this scale and code quality.

This is about applications that are actually useful and safe, that belong to the user, and make computers fun again.

I witnessed maybe 2 years of work, get done in just about 60 minutes, that’s exactly what I always expected from AI.

As to the rest of the world, enjoy the CEOs getting baffled, and understand that AI needs a new language with more semantic information.
