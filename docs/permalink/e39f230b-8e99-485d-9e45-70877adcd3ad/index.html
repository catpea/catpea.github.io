Could 3D Code Editors Work?

6/18/2026

Lets get something out of the way first, AI is fuzzy, but so are you.

With that out of the way, Know that 3D code editors don’t exist yet.

People have built a code density visualizer, That shows something that looks like a city.

Others did some graphs, That you can uselessly turn around.

—

My life’s work is a tiny little thing, A visual programming language.

I have a beautiful text notation, My favor implementation uses EventEmitters.

My most successful version, Can swap AI modules.

And my greatest one, was an afterthought, AI made it as part of a UI toolkit.

—

But none of this moves me anymore, It is not a challenge.

So I have been bothering AI about, Futuristic user interfaces.

And just this afternoon, I went from flat subway maps.

To a fascinating 3D space station cluster, I use multiple visualizations.

Every good programming idea gets one, EventEmitter for example.

And then EventEmitters events, Get a concept map of where events go.

—

Space Stations instead of Subway Maps, With conceptual information.

And around the concepts, Rotate properties.

—

These orbitals as I called them, Are kind of the point.

It is their size, color, density, That visually reveals the objects.

I keep wanting to add 2D like features, I don’t like 3D, I keep getting lost.

But it is already clearly the right way, Certainly to navigate well made AI code.

—

Before I got into this. I built my 2D maps in VR.

I could fly around them, But that was not enough functionality.

A good system must work well on 2D screens, Before you interact with in VR.

—

As to code editing, selecting a code node, Just opens a code editor in the correct place.

From here, I would add 3d bookmarks, Relative to moving objects.

And then pull up to them, To return to whatever I was working on before.

—

Though I built this with AI in mind, Where AI is the code processor.

That creates all these maps, Based on its observations.

That turned out not to be needed, Walking the code is easy.

—

What the AI is good for here is, A kind of a trick.

Where you ask it to add something, Based on the way something else exists.

Even a local AI will make a coherent addition, That will expand the program.

Without the 3D map a human would be lost, There would be nothing to select or outline.

—

With AI doing most of the programming today, 3D code editors are the future.
